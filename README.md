# SmartHomeCenter
一个智能家庭中心服务器。

## 一、技术框架
    服务器软件使用Nodejs + MySql 技术。
## 二、软件平台
1. 使用Socket.io框架实现实时推送技术。
2. 使用express HTTP框架。
3. 使用mysql驱动访问mysql数据库。
4. 使用log4js 记录日志。

## 版本依赖：   
* nodejs v4.2.1  
* express v4.14.1  
* socket.io v1.7.2  
* mysql v2.13.0  
* log4js v1.1.0
